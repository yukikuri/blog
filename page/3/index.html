Ancia Ver1.9.119をリリースしています。更新内容は、次の通りとなります。 更新内容 add ActiveScriptの実行Versionで環境に応じた最新のJScriptで実行を追加、また規定の設定に Active Scriptの実行Versionが環境に応じて最新のVersionで実行します。IE9以降がインストールされていれば、高速なエンジン(JScript9.dll)でユーザスクリプトが実行されます。 fix youtubeでfullscreenが動作しないのを修正 報告ありがとうございます Youtube で Player でのフルスクリーンボタンが反応しないのが修正されています。報告ありがとうございました。 fix 起動時のタブ復元でcloselockを外しても復活するのを修正 ナビゲートロックだけしている状態で、タブを復元するとクローズロックを外していても復活する不具合を修正しています。 fix Google Chrome がインストールされていない場合、Chromeのブックマークを表示するとCrashするのを修正 報告ありがとうございます Google Chrome がインストールされていない状態で Google Chrome のお気に入りを表示するとクラッシュする不具合を修正しています。報告ありがとうございます。 fix Windows Storeでのアプリをゲットのボタンが動作しないのを修正 Windows ストア のページで、「アプリをゲット」ボタンが無反応である不具合を修正しています。 mod 前回終了時のタブ保存で、タブ情報が取得できない場合できるだけ情報を保存するように(その場合、ハイライト、戻る進む履歴、自動更新情報は失われます) mod IPC通信のタイムアウト時間を長めに マルチプロセスでタブの終了状態が取得できない場合の発生が起こる場合があり(こちらでは再現できず)、IPC通信のタイムアウトと取得失敗時でもできるだけタブ情報を保存するようにしています。 mod window.open()でタブを開く際のpopup block設定が効いていないのを修正 ポップアップブロックの設定が、window.open()でタブを開く場合に判定されていない(常にIEでの判定が行われる)不具合を修正しています。報告により試すと発見できました。報告ありがとうございました。 mod ローカルファイルやインターネットショートカットなどをタブバーにdropできるように 今更ですがタブにローカルファイルなどをドロップして開けるようにしました。 ダウンロードと更新一覧 ダウンロードはこちらより行えます。ダウンロード Ancia 1.9.119 更新一覧はこちらとなります。更新一覧 Ancia 1.9.119 不具合について もし不具合がある場合、ソフトウェアフォーラム までお願いします。

Ancia Chrome Ver0.5.4 リリース

2016年05月23日 online soft release
ブログの更新が遅れましたが、2016/5/2 に Ancia Chrome Ver0.5.4 をリリースしています。Blinkのバージョンは49のままですので、XPでも使用できます。 更新内容 add DNSエラーや404エラーなどで、Google Chromeと同様のエラーページを表示を行うように Google Chromeと同等のページエラー表示を可能にしています。 add mp4やmp3直リンを再生可能に mp4やmp3の直リンク表示(??.mp4)などの直接の再生を可能にしています。報告ありがとうございます。 add 新規タブでよく表示すページを表示するように add よく表示するページやHistory、Thumbnailを保存するように(現時点では、よく表示するページを新規タブで表示するのみをサポート) 表示したページの履歴などをローカルに保存するようにしています。新規タブでよく表示するページを表示するようにしてみました。 fix 特定のvideoタグでmedia fallbackが動作しないのを修正 videoタグでsourceの指定方法によりmp4などの再生が行われない場合を修正しています。 fix 特定のFormでsubmit時にcrashするのを修正 報告ありがとうございます 特定のフォームで送信すると、フォームのオートコンプリート関係でクラッシュする問題を修正しています。 fix pdfが表示できないのを修正 PDFが表示できていなかったのを、表示可能になりました。 ダウンロードと更新一覧 ダウンロードはこちらより行えます。ダウンロード Ancia Chrome 0.5.4 更新一覧はこちらとなります。更新一覧 Ancia Chrome 0.5.4 不具合について もし不具合がある場合、ソフトウェアフォーラム までお願いします。
