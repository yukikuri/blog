Ancia Ver1.9.127 リリース

2017年05月22日 online soft release

Ancia Ver1.9.127 をリリースしています。関連付けでアプリケーションが固まってしまう不具合などを修正しています。 更新内容は次の通りです。

更新内容

  • fix オプション設定から関連付けのためレジストリ設定を行うと固まるのを修正 報告ありがとうございます

    でアンチウイルス誤検知対策のため関連付けを行う動作を 1.9.125 で変更したのですが、関連付けのためレジストリ登録を行うと固まっていました。Debug バージョンでは発生せず Release バージョンで出る問題です。 今回固まる致命的な問題を修正しています。報告ありがとうございました。

  • mod これ以外のタブを閉じる, 選択リンクを開く(closeExcept.js, openSelectionLink.js)のアイコンを追加 作成ありがとうございます

    Blink 版では先に追加していますが、Trident 版でも作成していただいたスクリプトアイコンを追加しています。

  • fix タブがないときに新規タブを作成したとき、新規タブにフォーカスが設定されないのを修正
  • fix タブがシングルラインの時、タブ幅の設定が効いていないのを修正 報告ありがとうございます

    ほか細かなバグを修正しています。

作者環境では Windows10 で Creators Update を適用し今のところ特に問題なく動作しています。
