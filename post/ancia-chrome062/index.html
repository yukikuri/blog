Ancia Chrome Ver0.6.2 リリース

2017年05月22日 online soft release

Ancia Chrome Ver 0.6.2 をリリースしています。

PDF を表示したページでメニュー表示に致命的な問題があり修正しています。更新内容は次の通りです。

更新内容

  • fix pdfのページで右クリックを行うとcrashするため、pdfクリックメニューは非表示に(対応までの動作仕様) 報告ありがとうございます

    PDF を表示したページで右クリックなどメニューを表示するとプロセスがクリックし、ほかのタブもすべて落ちていました。 PDF ページではメニュー表示は非表示としています。本動作は対応まで仕様となります。

    PDF ページでのテキスト選択のコピーなどは、ショートカットキーCtrl+Cや、メニューバーの「編集」「コピー」が利用できます。

ダウンロードと更新一覧

更新一覧はこちらとなります。更新一覧 Ancia Chrome 0.6.2

不具合について

もし不具合がある場合、ソフトウェアフォーラム までお願いします。
