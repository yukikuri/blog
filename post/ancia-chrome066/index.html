NEWAncia Chrome Ver0.6.6 リリース

2017年10月23日 online soft release

Ancia Chrome Ver 0.6.6 をリリースしています。

主な更新内容は、Ancia Ver 1.9.131 の更新内容と、Blink エンジンの更新が主です。Chrome 61 で WebUSB 機能が追加されていますが既定で無効化しています。 更新内容は次の通りです。

更新内容

  • fix Windows10の環境で、ホバーしたとき非アクティブ ウィンドウをスクロールするがオンのときホイールすると、2回スクロールしてしまうのを修正

    Windows10の環境だけで起こる現象で、非アクティブウィンドウでのホイールスクロールする動作が原因で2回スクロールしてしまうを修正しています。

  • mod WebUSB機能をデフォルトで無効に(navigator.usbの非サポート)

    61で追加された WebUSB 機能を無効化しています。navigator.usb 自体を参照できなくしています。

  • fix アドレスバーでの貼り付け移動・貼り付け新規タブでアドレスの履歴に保存されないのを修正
  • fix フルスクリーンで、ハイライトをしている場合などで上部ツールバーが出ないのを修正
  • fix オプション設定のタブに関わるマウス動作を設定するとタブ背景でメニューが出なくなるのを修正 もしタブ背景でメニューが出ない場合は、タブ背景の動作で「規定値に設定する」をお願いします
  • mod オプション設定のpageHighDpiが無効の時のスケールファクター値で小数点以下の値を削除 もしスクロールできない現象が出た場合forumまで報告してもらえると助かります
  • mod mailtoリンクで新規タブが作成されると空タブができるのを抑止
  • fix オプション設定のマウス動作で旧設定が残っている場合、開けないのを修正

    ほか細かな不具合などを修正しています。もしタブの背景でメニューが出ない場合は、「規定値に設定する」ボタンでメニューが出るようになります。もしくは、メニューを表示に設定することでも出るようになります。

  • mod blink版Chrome/61.0.3163.100に更新

    本日(2017/10/23)の最新バージョンは 62.0.3202.62 ですが、更新に間に合いませんでした。62.0.3202.62 への更新はさらに時間がかかる恐れがあるため更新済みの 61.0.3163.100 でリリースしています。

  • add バージョン情報chrome://newtab/#versionで、自動で更新を確認するかの設定を追加
  • add ツールバーで最新に更新ボタンを長押ししたとき、「ハード読み込み」「すべてのタブ更新」が行えるメニューを出すように
  • add アプリケーションの更新でコピー状況がわかるように表示するように
  • fix アプリケーションの更新(Updater)で、動作中のプロセスが256を超えているとき、アプリケーションの更新が終わらないのを修正
  • mod Windows10でウインドウタイトルの文字色をできるだけWindowsに合わせるように
  • mod 貼り付け移動やテキストをURLで開くスクリプトで、chromeプロトコルをサポート

    Ancia Ver 1.9.131 の更新内容と同等の内容を含めています。また、Ancia Ver 1.9.129の更新内容での favicon 表示問題などの対応も含まれています。詳しい更新内容は該当記事を参照願います。
