Ancia Ver1.9.119 リリース

Ancia Ver1.9.119をリリースしています。更新内容は、次の通りとなります。

更新内容

  • add ActiveScriptの実行Versionで環境に応じた最新のJScriptで実行を追加、また規定の設定に

    Active Scriptの実行Versionが環境に応じて最新のVersionで実行します。IE9以降がインストールされていれば、高速なエンジン(JScript9.dll)でユーザスクリプトが実行されます。

  • fix youtubeでfullscreenが動作しないのを修正 報告ありがとうございます

    Youtube で Player でのフルスクリーンボタンが反応しないのが修正されています。報告ありがとうございました。

  • fix 起動時のタブ復元でcloselockを外しても復活するのを修正

    ナビゲートロックだけしている状態で、タブを復元するとクローズロックを外していても復活する不具合を修正しています。

  • fix Google Chrome がインストールされていない場合、Chromeのブックマークを表示するとCrashするのを修正 報告ありがとうございます

    Google Chrome がインストールされていない状態で Google Chrome のお気に入りを表示するとクラッシュする不具合を修正しています。報告ありがとうございます。

  • fix Windows Storeでのアプリをゲットのボタンが動作しないのを修正

    Windows ストア のページで、「アプリをゲット」ボタンが無反応である不具合を修正しています。

  • mod 前回終了時のタブ保存で、タブ情報が取得できない場合できるだけ情報を保存するように(その場合、ハイライト、戻る進む履歴、自動更新情報は失われます)
  • mod IPC通信のタイムアウト時間を長めに

    マルチプロセスでタブの終了状態が取得できない場合の発生が起こる場合があり(こちらでは再現できず)、IPC通信のタイムアウトと取得失敗時でもできるだけタブ情報を保存するようにしています。

  • mod window.open()でタブを開く際のpopup block設定が効いていないのを修正

    ポップアップブロックの設定が、window.open()でタブを開く場合に判定されていない(常にIEでの判定が行われる)不具合を修正しています。報告により試すと発見できました。報告ありがとうございました。

  • mod ローカルファイルやインターネットショートカットなどをタブバーにdropできるように

    今更ですがタブにローカルファイルなどをドロップして開けるようにしました。

ダウンロードと更新一覧

ダウンロードはこちらより行えます。ダウンロード Ancia 1.9.119
更新一覧はこちらとなります。更新一覧 Ancia 1.9.119

不具合について

もし不具合がある場合、ソフトウェアフォーラム までお願いします。
