Ancia Chrome Ver0.6.0 リリース

2017年05月09日 online soft release

Ancia Chrome Ver 0.6.0 をリリースしています。

0.6.0 では、レンダリングエンジンを2017年5月現在の最新Verに更新、また、コンテンツ保護されたメディアの再生、終了時にクラッシュするバグ修正などを行っています。 更新内容は次の通りです。

このブログ記事が出る前に 0.6.0 へ更新した方へ

このブログ記事公開前の 0.6.0 配布ファイルで、タブを別ウインドウに切り離すとフォーカスが別ウインドウに移らないバグがありました。 配布ファイルをこのバグを修正したものに置き換えています(バージョンは 0.6.0.1)。

もしこのバグが気になるときは、お手数ですが配布ファイルをダウンロードし、AnciaChrome.exe だけをダウンロードしたものに上書きをお願いします。

更新内容

  • add Amazon ビデオや Netflix などコンテンツ保護された動画を再生できるように(コンテンツ保護技術 Widevine CDM のサポート)

    Amazon ビデオなどコンテンツ保護メディア再生に対応しています。Amazon ビデオと Netflix で再生できることを確認しています。 Widevine CDM のファイルは関係上同梱できないため、含めない形での対応を行っています。詳細は技術的なこととなるため省略します。

  • fix タブがないときなどのアプリケーション終了時、クラッシュするのを修正 報告ありがとうございます

    アプリケーション終了時、Blinkエンジン側の終了処理に問題が見つかり、正常に終了できるよう対応しています。報告ありがとうございました。

  • fix 高DPI環境でメニューの位置がずれるのを修正 報告ありがとうございます

    高DPI環境だとメニュー表示位置が高DPI表示のページ側の座標で表示していたのを修正しています。報告ありがとうございました。

  • mod 新規タブで前回終了時のタブなどを表示するように(タブリストを表示するように)

    Trident版で対応していた新規タブでタブリストを表示する対応をBlink版でも行っています。

  • mod blink版Chrome/58.0.3029.96に更新

    レンダリングエンジンを現在の最新バージョンに更新しています。

ダウンロードと更新一覧

ダウンロードはこちらより行えます。ダウンロード Ancia Chrome 0.6.0
更新一覧はこちらとなります。更新一覧 Ancia Chrome 0.6.0

不具合について

もし不具合がある場合、ソフトウェアフォーラム までお願いします。
