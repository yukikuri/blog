Ancia Chrome Ver0.6.5 リリース

2017年08月17日 online soft release

Ancia Chrome Ver 0.6.5 をリリースしています。

主な更新内容として、アドレスバーから Google Chrome と同様の履歴などの検索候補の表示を可能にしています。 更新内容は次の通りです。

更新内容

  • add Google Chromeと同等のアドレスバーから履歴検索や検索候補を表示などを可能に

    Google Chrome はアドレスバーから検索候補や履歴などからサイトの候補を表示できます。今回の対応で同様の動作をサポートしています。残念ながら検索候補での検索は Google 固定で変更はできません。Google 以外で検索候補が表示できるかは未知のため、検索エンジンの変更はサポートできるかはわかりません。

  • fix ニコニコ動画など検索バーやメニューなどから検索で%を含むキーワードが検索できない場合があるのを修正 報告ありがとうございます
  • fix 検索履歴からキーワードを削除すると画面のactive状態が変わるのを変わらないように修正
  • fix ハイライトを行った後フルスクリーン表示を行い、ハイライトを非表示にしフルスクリーンを解除すると無意味な描画がウインドウ上部に出るのを修正
  • mod アドレスバー履歴表示でのボタンを一部変更(ローカルホームボタンの変更)
  • mod 最小化状態で開発者ツールを表示すると開発者ツールが表示されないのを表示されるように
  • mod 開発者ツール表示後、再度表示したとき、ウインドウをActiveに

    開発者ツールを表示した後、再度開発者ツールを表示してもActiveにならない、最小化のとき表示されないといったことへの対応を行っています。

ダウンロードと更新一覧

ダウンロードはこちらより行えます。ダウンロード Ancia Chrome 0.6.5
更新一覧はこちらとなります。更新一覧 Ancia Chrome 0.6.5

不具合について

もし不具合がある場合、ソフトウェアフォーラム までお願いします。
