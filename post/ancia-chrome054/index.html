Ancia Chrome Ver0.5.4 リリース

ブログの更新が遅れましたが、2016/5/2 に Ancia Chrome Ver0.5.4 をリリースしています。Blinkのバージョンは49のままですので、XPでも使用できます。

更新内容

  • add DNSエラーや404エラーなどで、Google Chromeと同様のエラーページを表示を行うように

    Google Chromeと同等のページエラー表示を可能にしています。

  • add mp4やmp3直リンを再生可能に

    mp4やmp3の直リンク表示(??.mp4)などの直接の再生を可能にしています。報告ありがとうございます。

  • add 新規タブでよく表示すページを表示するように
  • add よく表示するページやHistory、Thumbnailを保存するように(現時点では、よく表示するページを新規タブで表示するのみをサポート)

    表示したページの履歴などをローカルに保存するようにしています。新規タブでよく表示するページを表示するようにしてみました。

  • fix 特定のvideoタグでmedia fallbackが動作しないのを修正

    videoタグでsourceの指定方法によりmp4などの再生が行われない場合を修正しています。

  • fix 特定のFormでsubmit時にcrashするのを修正 報告ありがとうございます

    特定のフォームで送信すると、フォームのオートコンプリート関係でクラッシュする問題を修正しています。

  • fix pdfが表示できないのを修正

    PDFが表示できていなかったのを、表示可能になりました。

ダウンロードと更新一覧

ダウンロードはこちらより行えます。ダウンロード Ancia Chrome 0.5.4
更新一覧はこちらとなります。更新一覧 Ancia Chrome 0.5.4

不具合について

もし不具合がある場合、ソフトウェアフォーラム までお願いします。
