Ancia Ver1.9.129 リリース

2017年09月19日 online soft release

Ancia Ver1.9.129 をリリースしています。既定のブラウザによりリンクバーやサイドバーで favicon が表示できない不具合などが修正されています。 更新内容は次の通りです。

更新内容

  • fix 特定のブラウザが規定の場合、規定のブラウザアイコンが優先され、お気に入りバーなどでfaviconが表示されないのを修正 報告ありがとうございます

    Edgeなどが規定のブラウザのとき、リンクバーやサイドバーで favicon が正常に表示されない不具合を修正しています。この問題は次回更新のBlink版にも反映されます。報告ありがとうございました。

  • fix 一度タブグループとなったタブから新規タブを開くと新規タブの追加位置設定がおかしくなることがあるのを修正 報告ありがとうございます

    タブの追加位置が設定と違った動作がする場合があり修正しています。フォーラムへの報告ありがとうございました。

  • fix 特定のページでfaviconが表示されないのを修正(アプリでのリクエストヘッダに関わる問題、表示されないサイトの例: https://wowma.jp/) 報告ありがとうございます

    favicon を取得する際のリクエストヘッダにより正常に favicon が返却されない場合がある問題を修正しています。報告ありがとうございます。

  • fix 起動時にブラウザ部分が空白になる現象が出ることがあるのを修正

    以前報告があったと思うのですが、起動時にブラウザ部分が空白となる現象が再現しました。マルチスレッド関係の問題で発生しないように対応しています。

  • mod 応答なしの現象が出るため、Windows10環境ではALT＋TAB動作対応はオプション設定testFeatureが有効の場合のみ動作するように

    1.9.128 で Windows10 限定でALT+TAB 動作修正を行いましたが固まる現象が出てしまいました。オプション設定の testFeature が有効の場合のみ ALT+TAB 対応が動作する対応を行っています。ALT+TAB が2回押す必要がある動作は対応が難しく仕様となります。

  • mod リンクバーなどインターネットショートカット(.url等)のアイコン表示で、favicon指定があってもfaviconが表示できない場合、できるだけfaviconを表示するように
  • mod sqlite3.20.1に更新

    他sqlite3の更新などを行っています。
