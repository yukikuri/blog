Ancia Chrome Ver0.6.3 リリース

2017年06月14日 online soft release

Ancia Chrome Ver 0.6.3 をリリースしています。

更新内容は少なく主にBlinkエンジンの更新や細かなバグ修正です。更新内容は次の通りです。

更新内容

  • add オプション設定にオートフィルの有効の設定追加(設定名:autofill、既定有効)

    フォームへのオートフィルを使用するかしないかを設定可能にしています。デフォルトは使用するです。

  • fix 検索バーでの履歴メニューを表示し、さらにメニューを表示すると履歴メニューが閉じない場合があるのを修正
  • fix アニメーションgifなどの再生ページで右クリックメニューの動作がおかしいことがあるのを修正
  • fix オプション画面でダウンロードなどのロゴ画像が表示されないことがあるのを修正
  • mod Windows10以外でインライン検索にフォーカスがあるときページでホイールスクロールできないのを修正

    細かなバグを修正しています。メニューの動作がおかしい件は、動画の再生ページだと、今回の対応で再生部分が描画されない影響が出ています。描画については対応を考えています。

  • mod blink版Chrome/59.0.3071.86に更新

    レンダリングエンジンの更新を行っています。Google Chrome 59での更新内容は、Web制作者は使わないと損！Chrome 59でアップデートされたデベロッパーツールの便利な新機能 | コリスが参考になります。 フルサイズのスクリーンショットをキャプチャができないのですが、対応したいところです。

ダウンロードと更新一覧

ダウンロードはこちらより行えます。ダウンロード Ancia Chrome 0.6.3
更新一覧はこちらとなります。更新一覧 Ancia Chrome 0.6.3

不具合について

もし不具合がある場合、ソフトウェアフォーラム までお願いします。
